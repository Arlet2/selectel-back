## Deploy

### Requirements:
* #### git
* #### Docker
* #### docker-compose
### To run app:
```shell script
# Clone project
$ git clone https://github.com/Arlet2/selectel-back.git
$ cd selectel-back/         
# Run bot with docker-compose (prod)
$ sudo docker-copmose -f docker-compose.yaml up --build -d
```
